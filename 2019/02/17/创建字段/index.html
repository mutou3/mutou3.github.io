计算字段

计算字段

  1. 含义 储存在数据库中的数据一般不是应用程序所需要的格式。计算字段并不实际存在于数据库表中，计算字段是运行在select语句内创建的。==字段==基本上就是列的意思。

  2. 拼接字段 把两个列拼起来使用concat（）函数。(mysql的不同之处，多数的数据库使用+或者||lai pinjie 而mysql使用concat（）函数)

  3. 使用别名 别名用as关键字赋予。

  4. 执行算数计算

    算数操作符 | 说明
    —–|——-
    +| 加
    -|减
    *|乘
    /|除
